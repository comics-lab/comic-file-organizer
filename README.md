# comic-file-organizer

Scan/rename/import: convert CBR→CBZ, rename per Mylar rules, move to filestore, update Mylar3.

## Quickstart
```bash
python3 -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
```
